不同的引用方式

在JAVA8中存在四种方法引用

引用方式 语法
引用静态方法 ContainingClass::staticMethodName
引用特定对象的实例方法 containingObject::instanceMethodName
引用特定类型的任意对象的实例方法 ContainingType::methodName
引用构造器 ContainingClass::new

引用静态方法

如果一个对象存在静态方法，那么可直接通过 ContainingClass::staticMethodName（即：类名::方法名）进行调用。

引用特定对象的实例方法

对于特定对象的实例方法，我们可以通过 containingObject::instanceMethodName（即：实例名::方法名）进行调用。

引用特定类型的任意对象的实例方法

特定类型指非自定义类型，如String，Integer等， 对于这种类型的实例方法我们可以通过

ContainingType::methodName（即：类型名称::方法名）进行调用。

引用构造器

这种引用和静态方法引用非常相似，我们可以通过 ContainingClass::new（即：类名::new）进行调用。


以上就是关于方法引用的一些基本介绍，总结自官方文档。
